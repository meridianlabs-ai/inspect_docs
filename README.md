# Inspect Docs

Inspect Docs helps you create documentation websites for Python packages using [Quarto](https://quarto.org):

- Author articles using all the features of standard Quarto markdown.
- Generate API reference pages from your source code via [griffe](https://mkdocstrings.github.io/griffe/).
- Automatically turn inline code references (e.g. ``` `my_function()` ```) into links to the right reference page.
- Generate an [llms.txt](https://llmstxt.org) and Markdown doc pages so LLMs can ingest your site.

To learn more about using Inspect Docs, vist the website at https://meridianlabs-ai.github.io/inspect-docs
